## Lessons Learned
- **Kimi K2.5 can't do tool calls via OpenRouter.** Broken JSON, spaces in tool names, thinking conflicts. Don't use it for tool-heavy tasks.
- **Always set `thinking: "off"`** for non-Anthropic models on cron jobs.
- **Always verify `enabled: true`** when creating new cron jobs ‚Äî don't assume default is enabled.
- **Cron `run` action only works if "due"** ‚Äî use `sessions_spawn` for manual triggers when nextRunAtMs is in the future.
- **Grok not allowed for sub-agent spawns** ‚Äî only works in cron payloads. Sub-agents fall back to default model (Opus).
- **Brave Search Free plan = 1 req/sec** ‚Äî don't fire multiple search-heavy agents simultaneously.
