# Onchain Research Tools — Tier List
Source: @stacy_muur https://x.com/stacy_muur/status/2020774481032335741

## S-Tier (Professional Edge)
- **Nansen** (nansen.ai) — Smart money tracking, wallet labels, best onchain intelligence (paid)
- **Arkham** (arkham.ai) — Free Nansen alternative, good enough for 90%+ of traders
- **Glassnode** (glassnode.com) — Bitcoin/Ethereum onchain metrics, institutional standard

## A-Tier (Essentials)
- **CryptoQuant** (cryptoquant.com) — Exchange flows, whale movements, derivative data
- **DefiLlama** (defillama.com) — TVL, DEX volumes, token unlocks, yields — all free
- **Dune** (dune.com) — Custom dashboards for any protocol (we already use this!)
- **Coinglass** (coinglass.com) — Funding rates, liquidations, long/short ratios

## B-Tier (Solid Tools)
- **Messari** (messari.io) — Research-grade data (paywall not worth it per Stacy)
- **Artemis** (artemis.xyz) — Clean L1/L2 comparisons, app analytics
- **CoinGecko** (coingecko.com) — Price tracking, quick checks
- **CoinMarketCap** (coinmarketcap.com) — Same as CoinGecko, heavier
- **CryptoRank** (cryptorank.io) — ATH tracking, category performance, new listings

## Key Insight
"Traders overpay for tools they don't need. The free alternatives are more than enough." — Stacy Muur
